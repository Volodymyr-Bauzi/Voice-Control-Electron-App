Place your Vosk speech recognition model here.

Download the appropriate model from: https://alphacephei.com/vosk/models

Recommended models:
- vosk-model-small-en-us-0.15 (small, ~40MB)
- vosk-model-en-us-0.22 (medium, ~1.8GB)

After downloading, extract the model files to this directory.
The directory structure should look like:
/resources/models/vosk/vosk-model-small-en-us-0.15/
  - am/
  - conf/
  - graph/
  - ivector/
  - rescore/
  - README
  - etc.

For better performance with non-native speakers, consider using a larger model.
